Software To Install After Installing Windows 10

After installing Windows 10, some neccessary software should be installed. Luckily, there aren‚Äôt as many settings to do as Ubuntu.

Install Windows

This is the Step 0. Windows 10 can be downloaded officially from Microsoft, choose Windows 10 Pro edition while installing. However, I prefer the LTSC (Long Term Servicing Channel) edition. Windows 10 Enterprise LTSC 2021 can be downloaded here, here is an ed2k link for it too but I don‚Äôt think there are enough seeders. Make sure not to download the evaluation edition from Microsoft official website.

Verify the checksum after download.

Use Rufus to create a bootable USB drives.

After installation, activate the system by KMS server provided by 03k.

Firefox ESR

Firefox is my favourate web browser and I prefer the ESR (Extended Support Release) edition because I don‚Äôt like updates. I may stay at the 78.15.0esr version.

Chinese IME

  • Rime

If you are just a Pinyin IME user, then the build-in Microsoft Pinyin IME will be enough. Sogou Pinyin IME is also another popular Chinese IME.

Windows Photo Viewer

Photos replaced Windows Photo Viewer as the default and only photo viewer in Windows 10, I prefer the old Windows Photo Viewer. Import the Registry file provoded by Shawn Brink from TenForums then Windows Photo Viewer will be back.

Microsoft Office

Microsoft Office can be downloaded by using the official Office Deployment Tool and Office Customization Tool. For Chinese residents, I will recommend the unofficial Office Tool Plus (or at Shangdong University Mirror) to get faster download speed. As for the edition, I will recommend Office Professional Plus 2019 - Volume License, then activate it by 03k‚Äôs KMS server.
